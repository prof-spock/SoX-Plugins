Detailed Description

The GenericNumber specification and body defines a template for wrapper objects for elementary numeric types.

Such an object just contains a single internal variable whose type is some kind of elementary type (int, long, double, ...).

An important benefit of this approach is that all free-floating functions on the elementary type can be gathered into a single class.

Very important is that the wrapper type abstraction does not impose a runtime penalty.
