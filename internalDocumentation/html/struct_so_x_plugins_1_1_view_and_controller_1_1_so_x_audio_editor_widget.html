Detailed Description

A SoXAudioEditorWidget object models an audio editor widget (a slider or a combo box) in a SoX audio editor for editing a single discrete value parameter; within the editor the widgets are layout out in a vertical manner as rows; note that there is no subclassing for comboboxex or sliders: all processing is done in a single class.
