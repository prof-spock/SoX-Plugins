Namespaces

 SoXPlugins
 The package SoXPlugins provides all types for the DAW plugins emulating the command-line audio processor SoX; the connection to the plugin host is abstracted via the JUCE framework, the effect implementation itself is completedly self-contained.
 
 SoXPlugins::ViewAndController
 The package ViewAndController provides classes for the interconnection to the enclosing plugin framework (abstracted via the JUCE framework): a SoXAudioProcessor connects to the plugin host, the SoXAudioEditor is the UI component and interacts with the user and consists of SoXAudioEditorWidgets like sliders and comboboxes to show and get data.
