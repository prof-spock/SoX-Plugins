Detailed Description

The SoXPhaserAndTremolo_AudioEffect body implements the SoX phaser and tremolo modulation effects that have similar parameters and hence are combined into a single plugin.

The original algorithms can be found in the SoX repository in the files

  • "phaser.c" by Juergen Mueller and Sundry Contributors with the code for the phaser effect
  • "synth.c" by Carsten Borchardt and Rob Sowell with the code for the synth base effect, and
  • "tremolo.c" by Rob Sewell with the code for the (wrapper) tremolo effect.
