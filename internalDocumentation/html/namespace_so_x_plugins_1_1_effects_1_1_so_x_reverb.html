Detailed Description

The package SoXReverb provides classes for the SoX reverb audio effect.

The original algorithms can be found in the SoX repository in the file

  • "reverb.c" by Rob Sewell with the code for the reverb effect based on freeverb by Jezar at Dreampoint.
