 SoXPlugins
 The package SoXPlugins provides all types for the DAW plugins emulating the command-line audio processor SoX; the connection to the plugin host is abstracted via the JUCE framework, the effect implementation itself is completedly self-contained.
 
 SoXPlugins::BaseTypes
 The package BaseTypes provides the elementary types used for the SoX plugins; those are primitive types (like strings or reals), containers (like generic tuples or matrices) and modules for assertion and logging management.
 
 SoXPlugins::BaseTypes::StringUtil
 The StringUtil package provides several string routines not included in the standard C++ library (like, for example, prefix calculation, whitespace stripping etc).
