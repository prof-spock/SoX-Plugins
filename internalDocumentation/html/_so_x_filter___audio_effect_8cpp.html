Namespaces

 SoXPlugins
 The package SoXPlugins provides all types for the DAW plugins emulating the command-line audio processor SoX; the connection to the plugin host is abstracted via the JUCE framework, the effect implementation itself is completedly self-contained.
 
 SoXPlugins::Effects
 The package Effects provides types for the different audio effects themselves; the currently supported SoX effects are compander, filter, gain, overdrive, phaser/tremolo and reverb.
 
 SoXPlugins::Effects::SoXFilter
 The package SoXFilter provides classes for the SoX audio effects for IIR filters; those are allpass, band, bandpass, bandreject, bass, biquad, equalizer, highpass, lowpass and treble filters that are combined into a single plugin.
