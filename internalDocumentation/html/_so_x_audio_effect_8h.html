 SoXPlugins
 The package SoXPlugins provides all types for the DAW plugins emulating the command-line audio processor SoX; the connection to the plugin host is abstracted via the JUCE framework, the effect implementation itself is completedly self-contained.
 
 SoXPlugins::Effects
 The package Effects provides types for the different audio effects themselves; the currently supported SoX effects are compander, chorus/echo/echos, filter, gain, overdrive, flanger/phaser/tremolo and reverb.
