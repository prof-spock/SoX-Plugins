Detailed Description

An _EffectDescriptor_PHTR object is the internal implementation of a effect descriptor type for phaser and tremolo; both effects share parameters and have similar implementations: hence they can be combined.

For the tremolo the phaser parameters are assumed as follows: delay is 0s, inGain and outGain are 1.0, the decay is not relevant.

For the phaser the tremolo parameter depth is not relevant and ignored.
