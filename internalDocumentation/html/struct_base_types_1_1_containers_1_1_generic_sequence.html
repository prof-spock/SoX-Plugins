Detailed Description

template<typename T, String elementToString = nullptr>
struct BaseTypes::Containers::GenericSequence< T, elementToString >

A GenericSequence is a generic sequence type of values with zero-based arbitrary indexed access to positions in the sequence.

Indexing starts at zero and is consecutive. Sequences also allow duplicate elements.

If the printing of the sequence shall be possible the elementToString function must be provided.
