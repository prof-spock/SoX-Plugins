INLINE Radians WaveForm::phaseByTime ( IN Real  frequency,
IN Real  timeOffset,
IN Real  currentTime 
)
static

Calculates time locked delta phase for position currentTime from timeOffset and frequency; this gives the phase shift within a wave form oscillating with frequency being at phase 0 for timeOffset for the time given by currentTime, e.g.

for a frequency of 0.25Hz when currentTime is timeOffset+1s, the phase shift is pi/4

Parameters
[in]frequencyfrequency of wave form
[in]timeOffsettime offset of wave form
[in]currentTimecurrent time for phase to be calculated

INLINE void WaveForm::set ( IN Real  length,

IN Real  maximumValue,
IN Radians  phase,

Updates waveform to length base points with a wave form given by kind; wave form has minimumValue and maximumValue as bounds; phase specifies the offset in the waveform and hasIntegerElements tells whether data has to be rounded to integer; note that the length is real, that means, albeit the internal representation is an array of samples, the advance operation takes care of the accumulated indexing error and corrects is as soon as it exceeds one sample position.

Parameters
[in]lengththe count of base points
[in]kindthe kind (sine or triangle)
[in]minimumValuethe minimum y-value in waveform
[in]maximumValuethe maximum y-value in waveform
[in]phasethe initial phase (in radians)
[in]hasIntegerValuestells whether waveform only may have integer values as y-values
