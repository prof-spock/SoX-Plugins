Detailed Description

The package SoXFilter provides classes for the SoX audio effects for IIR filters; those are allpass, band, bandpass, bandreject, bass, biquad, equalizer, highpass, lowpass and treble filters that are combined into a single plugin.

The original algorithms can be found in the SoX repository and the file "biquads.c" by Rob Sewell.
