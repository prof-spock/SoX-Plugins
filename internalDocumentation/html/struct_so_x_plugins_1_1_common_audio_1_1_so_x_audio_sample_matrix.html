Detailed Description

A SoXAudioSampleMatrix object is a list of ring buffers for audio samples with selection by index and left or right rotation on the individual buffers.

Makes a list of sample queues for channelCount channels with arbitrary length; depending on hasTwoQueuesPerChannel either one or two queues are provided per channel; initial sample queue length is sampleQueueLength
