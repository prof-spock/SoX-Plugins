Detailed Description

template<typename Type, typename ElementaryType>
struct BaseTypes::Primitives::GenericNumber< Type, ElementaryType >

A GenericNumber is a generic number type that defines a wrapper around elementary numeric types (like int, long, etc.); * such a type contains a single internal variable of this elementary type and provides restricted access to it.

An important benefit of this approach is that all free-floating functions on the elementary type can be gathered into a single class.

Very important is that the wrapper type abstraction does not impose a runtime penalty.
