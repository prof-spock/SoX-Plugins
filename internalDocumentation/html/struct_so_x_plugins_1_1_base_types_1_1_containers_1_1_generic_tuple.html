Detailed Description

template<typename T, size_t elementCount, String elementToString = nullptr>
struct SoXPlugins::BaseTypes::Containers::GenericTuple< T, elementCount, elementToString >

A GenericTuple is a generic tuple type of values with zero-based arbitrary indexed access to positions in the tuple and a fixed length.

Indexing starts at zero and is consecutive. Tuples also allow duplicate elements.

If the printing of the tuple shall be possible the elementToString function must be provided.
